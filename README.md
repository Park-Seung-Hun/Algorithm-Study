## Rule

1. 시간은 `수요일 요일 오후 9시반`
2. 난이도는 `실1~골1 중 택` , 한회당 **4문제씩**.
3. 리뷰 풀어온 문제 중에 1~2문제 선택해서 진행.
4. 주마다 알고리즘별로 나눠서 문제 풀기 `그래프, 구현, dp, 브루트포스, 정렬, 탐색, 그리디, 문자열 등`
5. 파일 명: `문제번호_문제명` 형식으로 작성
6. commit message: `이름_해당 스터디 날짜` 형식으로 작성
6. 코드 내 주석(간단한 풀이 내용)
